实现的效果类似于只执行当前线程，其他线程暂时不运行

多线程调试命令：

需要在调试控制台的命令行中执行

-exec info threads
#展示当前所有线程
-exec thread 1
#切换至线程1
-exec thread 2
#切换至线程2
-exec set scheduler-locking on
#锁定当前线程
-exec set scheduler-locking off
#取消锁定当前线程
